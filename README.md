# ntpc_project
## An android application along with a web portal for the gate-pass generation and appointment facility to the visitors of NTPC power plant. This project is capable to help the CISF officers to generate and validate gate pass of visitors. Also visitors can take and confirm their appointments in advance by using the application or web portal.
